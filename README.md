## 🚀 Roadmap (Clean Order)

1. **HTML**
   - Structure: header, sections, footer

2. **CSS**
   - Styling: colors, fonts, spacing, layout
   - Flexbox, Grid, Media Queries
   - Google Fonts, Font Awesome

3. **Bootstrap (or Tailwind)**
   - Responsive grid system
   - Components: navbar, cards, modals

4. **JavaScript**
   - ES6+ basics
   - DOM manipulation, events
   - LocalStorage, form validation

5. **React**
   - Components, props, state
   - Hooks (useState, useEffect)
   - React Router
   - UI libraries (Material‑UI, Tailwind)

6. **Node.js + Express**
   - Server setup
   - REST APIs
   - Middleware
   - Nodemon, Postman

7. **MongoDB**
   - CRUD operations
   - MongoDB Atlas
   - Mongoose
   - Compass

8. **MERN Integration**
   - React frontend + Node/Express backend + MongoDB database
   - Axios/Fetch API
   - JWT/Passport.js (authentication)
   - Git/GitHub (version control)
   - Deployment: Netlify, Render/Heroku, MongoDB Atlas
